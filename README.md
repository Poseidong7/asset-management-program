# asset-management-program
An asset management program created using AI.


주요 기능
1. AI 채팅
 - 대화를 통해 현재 자산이 얼마 남았는지, 지출은 어느 정도인지 계산 가능.
 - 3가지 페르소나를 통해 다채로운 재미 탑재
 - 채팅으로 자신의 수입, 지출 등을 대화형식으로 적으면 자동으로 각 카테고리에 맞게 저장.

2. 실시간 주식/코인 추적
 - 내가 보유한 주식과 코인의 실시간 수익률 계산
 - 환율 자동 적용해 원화 가치로 환산 해 출력




설치 및 실행 방법
1. 프로젝트 다운로드
git clone https://github.com/당신의_아이디/DONGVIS.git
cd DONGVIS


2. 필수 라이브러리 설치
pip install -r requirements.txt


3. API 키 설정
 1. app.py 파일 열기
 2. os.environ["GOOGLE_API_KEY"] = "여기에_당신의_구글_API키를_넣으세요"
 

4. 터미널에 아래 명령어를 입력하세요
streamlit run app.py
